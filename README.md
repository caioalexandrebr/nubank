### Installing dependencies

```bash type=highlight
$ yarn install

// or

$ npm install
```

### Installing pods

```bash type=highlight
$ cd ios/
$ pod install
```

### Starting server

```bash type=highlight
$ react-native start --reset-cache

// or

$ yarn start
```

### Starting emulator
#### iOS
```bash type=highlight
$ react-native run-ios

// or

$ yarn ios
```
#### Android
```bash type=highlight
$ react-native run-android

// or

$ yarn android
```
